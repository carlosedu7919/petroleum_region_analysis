1.Introduo

A explorao de petrleo uma atividade estratgica e de alto investimento, exigindo decises precisas para maximizar o retorno financeiro e minimizar riscos. Neste contexto, a empresa OilyGiant busca identificar as regies mais promissoras para o desenvolvimento de novos poos de petrleo, utilizando dados geolgicos coletados em trs reas distintas.

O objetivo deste projeto aplicar tcnicas de Cincia de Dados para analisar os dados disponveis, construir modelos preditivos e, por meio de simulaes estatsticas, avaliar o potencial de lucro e o risco associado a cada regio. A abordagem adotada envolve:

  • Anlise exploratria dos dados de cada regio;
  • Treinamento de modelos de regresso linear para prever o volume de reservas em novos poos;
  • Seleo dos poos mais promissores com base nas previses do modelo;
  • Clculo do lucro potencial considerando as restries de oramento e receita;
  • Avaliao do risco de prejuzo utilizando a tcnica de bootstrapping.

Ao final, ser feita uma comparao entre as regies, levando em conta tanto o lucro mdio quanto o risco de perdas, para recomendar a melhor estratgia de investimento para a empresa. O projeto demonstra, assim, como a Cincia de Dados pode apoiar decises crticas em setores de grande impacto econmico.

Contexto do Desafio e Critrios de Seleo

Ao explorar cada regio, um estudo realizado em 500 pontos geolgicos. Aps a anlise, os 200 melhores pontos (aqueles com maior volume previsto de reservas) so selecionados para o clculo do lucro potencial.

  • Oramento: O desenvolvimento dos 200 poos selecionados deve respeitar um limite de 100 milhes de dlares.
  • Receita: Cada barril de petrleo bruto gera uma receita de 4,5 dlares.
  • Unidade de Produto: Como o volume de reservas est em milhares de barris, a receita de uma unidade de produto 4.500 dlares.
  • Risco: Aps a avaliao dos riscos (simulaes estatsticas), apenas as regies com risco de perdas inferior a 2,5% so consideradas elegveis.
  • Seleo final: Entre as regies que atendem ao critrio de risco, deve-se escolher aquela com o lucro mdio mais alto.
  • Observao: Todos os dados utilizados so sintticos e no contemplam detalhes contratuais ou caractersticas especficas dos poos.

Esse conjunto de regras garante que a deciso de investimento seja baseada em critrios quantitativos claros, equilibrando retorno esperado e risco financeiro.

def treinar_e_avaliar (X_train_scaled, X_valid_scaled, y_train, y_valid, n):
    model = LinearRegression()
    model.fit(X_train_scaled, y_train)
    y_pred = model.predict(X_valid_scaled)
    mse = mean_squared_error (y_valid, y_pred)
    rmse = np.sqrt(mse)
    media_predita = np.mean(y_pred)
    
    print ( f'RMSE (Regio {n}): {rmse:.2f}')
    print ( f'Volume mdio previsto (Regio {n}): {media_predita:.2f}')
    
    return mse, rmse, media_predita, model

def calcular_lucro_top200(X, df, model, scaler, nome_regiao, custo_total=100_000_000, receita_unidade=4500):
    # Padroniza as features
    X_scaled = scaler.transform(X)
    # Gera as previses e adiciona ao DataFrame original
    df['predicted'] = model.predict(X_scaled)
    # Seleciona os 200 melhores poos
    top200 = df.nlargest(200, 'predicted')
    # Calcula receita e lucro
    receita = top200['predicted'].sum() * receita_unidade
    lucro = receita - custo_total

    print(f'Lucro potencial ({nome_regiao}): ${lucro:,.2f}')
    print(f'Receita total ({nome_regiao}): ${receita:,.2f}')
    print(f'Volume total previsto ({nome_regiao}): {top200['predicted'].sum():,.2f} mil barris')

    return top200, receita, lucro

def bootstrap_risco_lucro(df, receita_barril, custo_por_poco, n_simulacoes=1000, n_pocos=200, n_total=500, nome_regiao="Regio"):
    import numpy as np

    def calculate_profit(sampled_df):
        best_well = sampled_df.sort_values(by="predicted", ascending=False)['product'].head(n_pocos)
        volume_well = best_well.sum()
        potential_profit = (volume_well * receita_barril) - (custo_por_poco * n_pocos)
        return potential_profit

    lucros_bootstrap = []
    for i in range(n_simulacoes):
        sampled_data = df.sample(n=n_total, replace=True, random_state=i)
        lucros_bootstrap.append(calculate_profit(sampled_data))

    lucros_bootstrap = np.array(lucros_bootstrap)
    prejuizo_count = np.sum(np.round(lucros_bootstrap, 2) <= 0)
    risco_prejuizo = prejuizo_count / n_simulacoes
    intervalo_confianca = np.percentile(lucros_bootstrap, [2.5, 97.5])
    lucro_medio = np.mean(lucros_bootstrap)

    print(f"\n{nome_regiao} - Lucro mdio: ${lucro_medio:,.2f}")
    print(f"{nome_regiao} - Intervalo de confiana 95%: ${intervalo_confianca[0]:,.2f} a ${intervalo_confianca[1]:,.2f}")
    print(f"{nome_regiao} - Probabilidade de prejuzo: {risco_prejuizo*100:.2f}%")

    return lucro_medio, intervalo_confianca, risco_prejuizo

8.Concluso

De acordo com a anlise de risco realizada via bootstrapping:

  • Apenas a Regio 1 elegvel para investimento, pois apresenta risco de prejuzo inferior ao limite de 2,5% estabelecido pelo projeto (1,8%) e tambm o maior lucro mdio entre as regies que atendem ao critrio.
  • As demais regies devem ser descartadas, j que o risco de prejuzo supera o limite permitido.

Esses resultados orientam a deciso estratgica, priorizando segurana financeira e retorno esperado conforme as regras do projeto.
